Install Terraform from Google

Steps to follow:
1. Download file
2. Extract to C or D :/Terraform
3. Open Enivornment Variables
4. System Variables > Find path > New > Add path
5. Open cmd > terraform --version

Next open VS Code
Install 3 extensions
1. Hashicorp Terraform
2. Hashicorp Sentinel
3. Hashicorp HCL

Make sure to keep all Terraform in one folder as they occupy lot of memory
